- 소인수분해의 어려움에 기반한 알고리즘
  소인수분해: A = BxC,  B와 C 찾기
- 두 큰 소수의 곱을 기반으로 함
- 높은 보안성
- 대칭키 암호화보다 느림
- 데이터의 안전한 전송, 인증, 디지털 서명등에 사용

- 공개키 : e, n
- 비밀키 : d

### 키생성
1. 2개의 큰 소수 p, q 선정
   n = p x q 
2. e는 pi(n) = (p-1) x (q-1) 과 서로소로 선정
3. e x d mod pi(n) = 1인 비밀키 d 계산

### 암호화/복호화
- 암호화
  e, n 받음
  E(M) = M^e(mod n) = C

- 복호화
  D(C) = C^d(mod n) = M
