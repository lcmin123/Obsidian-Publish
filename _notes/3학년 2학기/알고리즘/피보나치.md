## 재귀 알고리즘
F(n)
// n번째 피보나찌 수 f_n을 분할 정복 기법을 이용하여 계산한다
// 입력: 양의 정수 n
// 출력: n번째 피보나찌 수 $f{n}$

if (n≤1)  return 1
else return F(n-1)+F(n-2)

## 비재귀 알고리즘 (동적 계획)
F_DP(F[ ], n)
// n번째 피보나찌 수 f_n을 동적 계획을 이용하여 계산한다
// 입력: 양의 정수 n, 크기 n의 배열 F[0 .. n – 1]
// 출력: n번째 피보나찌 수 $f{n}$

F[0] = 1
F[1] = 1
for (i = 2; i ≤ n; i++)
	F[i] = F[i – 1] + F[i – 2]
return F[n]
